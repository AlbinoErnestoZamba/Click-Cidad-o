# CivicLink Angola 🇦🇴

## Descrição

O CLinkCidadão Angola é uma plataforma digital que promove a participação da juventude na governação local, permitindo reportar problemas, acompanhar projetos públicos e aumentar a transparência através de tecnologia.

## Funcionalidades

* Reportar problemas comunitários
* Votação e priorização de problemas
* Dashboard com mapa interativo
* Página de transparência com dados de orçamento
* Notícias simplificadas com apoio de IA
* Sistema inteligente que conecta problemas com projetos governamentais

## Diferencial

Utilizamos inteligência artificial para:

* Resumir notícias complexas
* Classificar problemas automaticamente
* Conectar problemas reais com planos orçamentais

## 🛠️ Tecnologias

* Frontend: React
* Backend: Python (Flask/FastAPI)
* Banco de dados: PostgreSQL

## Objetivo

Facilitar a participação cidadã e promover transparência na governação local em Angola.
